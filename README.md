# endometriose blog-post Header picture generator.

This leverages n8n as a "backend", together with [Pollinations.ai](https://pollinations.ai/), to generate images for blog-post headers, matching the overal website aestethic of [Endometriose.app](https://endometriose.app/).  
The website is hosted on [netlify](https://fluffy-chimera-3b83f4.netlify.app/)
